补充

如何上传文章

在上传文章的过程中又遇到了种种问题，总结一下

编写新文章
hexo n “新文章标题”
或
hexo new “新文章标题”

清除旧的数据
hexo clean

生成新的页面
hexo g
或
hexo generate

本地开启服务器，预览
hexo s
或
hexo serve

部署到github上
hexo d
或
hexo deploy

中文文章报错

搭建完后自然就想试一下手，笔者就简单的

hexo n “Hexo博客搭建成功！！！”
hexo clean
hexo g

进行到hexo g 这一步就报错了…
不死心的我硬是上传到了github上，结果

直接清库了！果然，头还是不能太铁啊。
然后我就查了一下，也没见有hexo不能上传中文文章的说法。
折腾了大概一个小时，气的我直接把除了_helloworld_这篇自带的文章全删了，接着

hexo n “restart!”
英文标题顺利上传…

没想到，再次试一次中文标题却ok了

这世界真是太奇妙啦！
